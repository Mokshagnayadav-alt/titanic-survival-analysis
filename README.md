# Titanic Survival Analysis 🚢

## 📌 Project Overview
This project analyzes the Titanic dataset and builds a machine learning model to predict whether a passenger survived or not.

The dataset is taken from the Kaggle Titanic competition.

---

## 📊 Problem Statement
The objective is to predict passenger survival based on features such as age, gender, passenger class, fare, and family size.

This is a binary classification problem:
- 0 = Did Not Survive
- 1 = Survived

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🔎 Steps Performed
1. Data Cleaning
2. Handling Missing Values
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building
6. Model Evaluation

---

## 📈 Key Insights
- Female passengers had higher survival rates.
- 1st class passengers survived more than 3rd class.
- Children had better survival probability.
- Passenger class and gender were strong predictors.
